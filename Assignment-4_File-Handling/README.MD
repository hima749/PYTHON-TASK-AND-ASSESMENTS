# Assignment - Functions with Loops

This assignment focuses on combining functions and loops in Python to solve repetitive and structured problems efficiently.

  Tasks Covered

* Implementing functions that use loops internally
* Performing iterative calculations using functions
* Processing lists and sequences using loops inside functions
* Creating reusable logic for repeated operations

 Concepts Used

* Function definition (`def`)
* for loops and iteration
* Parameters and return values
* Nested logic (functions + loops)
* Code reusability and modularity

 Objective

The objective of this assignment is to understand how combining functions and loops can simplify complex problems and improve code efficiency.

 File

* Jupyter Notebook containing all implementations

 Outcome

Gained practical experience in writing efficient and reusable Python code by integrating functions with looping constructs.
