# Assignment 2 - Logic Building Tasks

This assignment focuses on solving basic real-world problems using Python programming. It emphasizes logical thinking through the use of conditional statements, loops, and string operations.

📌 Tasks Covered

* User authentication using conditional statements
* Pass/Fail analysis based on student marks
* Data cleaning using string manipulation
* Message length analysis
* Error detection in text/log data

 🧠 Concepts Used

* if-else conditions
* for loops
* Lists and iteration
* String methods (`strip()`, `lower()`, etc.)
* Counting and filtering logic

## 🎯 Objective

The objective of this assignment is to strengthen problem-solving skills and understand how basic Python constructs can be applied to practical scenarios.
