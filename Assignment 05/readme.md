# Assignment 5 - Function-Based Problem Solving

This assignment focuses on solving real-world problems using a function-based approach in Python. It emphasizes writing modular, reusable, and structured code.

 Tasks Covered

* Breaking problems into smaller functions
* Performing calculations using functions
* Processing data using reusable logic
* Implementing multiple problem statements using functions
* Improving code readability and organization

 Concepts Used

* Function definition (`def`)
* Parameters and arguments
* Return values
* Code modularity
* Logical problem-solving

 Objective

The objective of this assignment is to develop the ability to design efficient solutions using functions and improve code reusability and clarity.

 File

* Jupyter Notebook containing all function-based implementations

 Outcome

Gained strong understanding of structuring programs using functions and solving problems in a modular way.
