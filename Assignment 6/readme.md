# Assignment 6 - Data Processing using Lists

This assignment focuses on handling and processing data using Python lists and basic data structures. It emphasizes applying logic to manipulate and analyze collections of data.

 Tasks Covered

* Working with lists and iterating over data
* Filtering and modifying list elements
* Performing calculations on datasets
* Applying conditions to structured data
* Solving real-world problems using list-based logic

 Concepts Used

* Lists and indexing
* for loops and iteration
* Conditional statements
* Data filtering and transformation
* Basic data analysis logic

 Objective

The objective of this assignment is to build the ability to process and analyze data efficiently using Python lists and logical operations.

 File

* Jupyter Notebook containing all implementations

 Outcome

Developed skills in handling structured data and applying logical operations to solve data-driven problems.
