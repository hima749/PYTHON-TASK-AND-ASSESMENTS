# Assignment  - 01

This assignment focuses on solving real-world problems using Python loops and conditional statements.

Tasks Implemented

1. User Login Check

* Validates username and password
* Demonstrates use of conditional statements (`if-else`)

 2. Pass / Fail Analyzer

* Analyzes student marks
* Counts number of pass and fail students using loops

3. Simple Data Cleaner

* Cleans raw data using string functions
* Removes spaces and converts text to lowercase

4. Message Length Analyzer

* Calculates length of messages
* Identifies messages longer than a specific limit

 5. Error Message Detector

* Counts occurrences of "ERROR" in logs
* Demonstrates loop-based counting logic

 🧠 Concepts Used

* if-else conditions
* for loops
* Lists
* String methods (`strip()`, `lower()`)
* Basic problem-solving logic

 🎯 Objective

To improve logical thinking and understand how Python can be used to solve real-world problems efficiently.
